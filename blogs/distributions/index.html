Understanding Distributions

and how to sample random values in PyTorch for initialization and training.

Why Distributions Matter

Distributions are fundamental to deep learning. We use them for weight initialization, data augmentation, and probabilistic modeling.

Key Concept: A distribution defines the probability of different values occurring. PyTorch provides easy ways to sample from various distributions.

Normal Distribution

The most common distribution in deep learning. Used for weight initialization (Xavier, He initialization).

import torch # Sample from normal distribution # mean=0, std=1 (standard normal) x = torch.randn(3, 3) print(x) # tensor([[-0.4125, 0.1681, -0.6782], # [ 1.2347, -0.2341, 0.5234], # [-0.1234, 0.8765, -0.3456]]) # Custom mean and std mean, std = 5, 2 x = torch.normal(mean, std, size=(3, 3))

Uniform Distribution

All values in a range are equally likely. Useful for random sampling within bounds.

import torch # Sample from uniform [0, 1) x = torch.rand(3, 3) # Custom range [min, max) x = torch.empty(3, 3).uniform_(-1, 1)

Summary

torch.randn() - Normal distribution (mean=0, std=1)
torch.rand() - Uniform distribution [0, 1)
torch.randint() - Random integers

Understanding distributions helps you initialize weights properly and add randomness where needed.
